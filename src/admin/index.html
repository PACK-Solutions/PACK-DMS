Search Query Help

Enter a JSON object to search documents by metadata (JSONB containment). Leave empty to list all documents.

Examples:

  • {"department": "Legal"}
  • {"author": "John Doe"}
  • {"tags": ["confidential"]}
  • {"department": "HR", "year": 2025}
